# Polygon-checker
Проверяет, находится ли маркер в границах полигона

### Запуск проекта
```
npm install
npm run dev
```

### Используемые библиотеки
- maplibre-gl
Для отображения карты и маркера

- @turf/turf
Для работы с полигоном

### Инструкция по использованию
- Маркер изначально уже указан на карте, задать новую позицию можно следующими способами:
1. Перетащить через drag-and-drop
2. Нажать ctrl + click в определенную точку карты
3. Воспользоваться строкой ввода адреса
В строке ввода адреса можно задать координаты через запятую в формате "ширина, долгота"(без кавычек).
Также можно ввести определенный регион и выбрать из предложенных вариантов

- Полигон добавляется по кнопке "Добавить полигон", создается в центре карты.
Новые вершины полигоны создаются по клике на карту, вершины также можно перетаскивать через dnd.
- По нажатию на кнопку "Проверить" отобразится alert с результатом проверки.
